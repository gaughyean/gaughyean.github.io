整理一些SWIFT的关键词

一、@escaping & @noescaping

用来修饰作为函数参数的闭包，escaping代表闭包不在函数执行的作用域中执行，noescaping反之，因此，noescaping是安全的。

二、@discardableresult

用在方法申明的前面，表面方法返回的结果可以不被使用，这样可以禁止编译警告。

三、defer

位于defer后面的闭包里面的代码将在离开defer所在的作用域的时候执行。 英文解释： The new defer keyword provides a safe and easy way to handle this challenge by declaring a block that will be executed only when execution leaves the current scope.

四、guard…else

尝试执行guard后面的代码，该代码的返回值是布尔类型，如果返回值是false，则执行else后面的代码，而这个else的代码必须离开guard代码所在的作用域。 英文解释： guard is a new conditional statement that requires execution to exit the current block if the condition isn’t met. Any new optional bindings created in a guard statement’s condition are available for the rest of the function or blockand the mandatory else must exit the current scope

五、mutating

可以使得struct 和 enum里面的成员变成可变

六、associatedtype

用于protocol里面，与遵循protocol的type对应。
